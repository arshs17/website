Automated Ball Shooter

Automated Ball Shooter

March, 2024

As part of the FRC 522 robotics team, I programmed an automated ball shooter using Java, integrating motion control algorithms and real-time sensor feedback to ensure precision targeting. I applied mathematical modeling to optimize shot trajectory and velocity, significantly enhancing performance in competitive robotics challenges. Additionally, I utilized OpenCV for real-time object detection and implemented PID control loops in Java to effectively track and adjust for moving targets. Python was used for data visualization and trajectory analysis, rounding out a robust, multi-language engineering solution.
